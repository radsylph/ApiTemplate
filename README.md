## How to Use
1. Clone the repository
```bash
git clone git@github.com:radsylph/ApiTemplate.git
```
2. Install the dependencies
```bash
npm install
```
3. Configure the basic environmental variable in a `.env` file:
```.env
NODE_ENV = development
PORT = 7338
```
4. Start the project.
```bash
npm run dev
```
